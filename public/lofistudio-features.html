LofiStudio

Espacio de trabajo productivo con música lofi, widgets inteligentes y un ambiente visual inmersivo.

Resumen

LofiStudio integra un sistema de widgets drag&drop, un reproductor de música con búsqueda en YouTube, temas adaptativos, fondos dinámicos y presets inteligentes para configurar todo en un clic. La disposición y preferencias se guardan automáticamente en el navegador.

Tecnologías: Next.js App Router, React 18, Tailwind, Shadcn/UI, React Grid Layout, Framer Motion, Three.js.

Editor de Layout

Activa el modo de edición para arrastrar y redimensionar widgets libremente.

  • Grid flexible con filas y columnas responsivas.
  • Prevención de colisiones.
  • Movimiento libre sin auto-compaction.
  • Persistencia automática del layout.
  • Panel “Add Widgets” con scroll y listado de widgets activos.
  • Límite seguro de capacidad para evitar saturar la pantalla.

Ctrl + E alterna el modo edición.

Widgets disponibles

Clock Hora en tiempo real.

Weather Pronóstico, ciudad y métricas.

GIF GIFs por categorías.

Tasks Crear, completar y eliminar.

Notes Editor rápido con guardado.

Quote Citas motivacionales.

Calendar Vista mensual.

Breathing Patrones guiados.

Dictionary Definiciones en inglés.

Timer Pomodoro configurable.

Regla: no se permiten duplicados de un mismo widget.

Estilo: todos los widgets usan glass consistente con el tema.

Reproductor de Música

  • Búsqueda en YouTube con sugerencias instantáneas.
  • Playlist persistente con selección rápida.
  • Controles: shuffle, repeat, anterior/siguiente, play/pause.
  • Mini reproductor y vista expandida.
Para búsqueda en YouTube se requiere YOUTUBE_API_KEY configurada en el entorno.

Ambient Mixer

Mezcla sonidos de ambiente para mejorar el enfoque:

  • Rain, Thunder, Wind, Forest, Fire, Café, Night.
  • Volúmenes independientes y mute global.
  • Persistencia de la mezcla.

Temas

Claro, Oscuro y Automático (se adapta a preferencias del sistema). Persistencia en localStorage y sincronización entre componentes.

Fondos

Gradient, escenas 3D (Room, Cafe), video de YouTube, imagen o video local cargado desde la PC.

Atajos de Teclado

Ctrl + E Editar Layout · Ctrl + , Settings · Ctrl + S Stats · Ctrl + L Logs · Shift + ? Ayuda · Alt + Z Zen Mode · Esc Cerrar modales

Settings

  • Seleccionar tema.
  • Ajustar duración Pomodoro.
  • Gestión de datos: exportar, importar y limpiar.
  • Gestión de fondo con selección de tipo y carga local de imagen/video.
  • Abrir Widget Manager desde Settings.

Estadísticas

Registro de sesiones Pomodoro completadas, tiempos y visualizaciones (heatmap, cards).

Gestor de Tareas

Creación/edición, completadas, y logs de actividad con persistencia local.

Cloud & Auth

Autenticación con Google (NextAuth). Integración opcional con Neon + Drizzle para sincronización y perfil.

APIs externas

  • YouTube Data API v3: búsqueda y playlist.
  • OpenWeather: clima y métricas.
  • Giphy: GIFs aleatorios por categoría.
  • Free Dictionary: definiciones.
Para activar estas integraciones, configura las variables de entorno: YOUTUBE_API_KEY, WEATHER_API_KEY, GIPHY_API_KEY.

Detalles de UX

  • Estética glassmorphism consistente.
  • Scroll atrapado en modales: al abrir un modal, el fondo no se desplaza.
  • Bienvenida y toasts informativos para onboarding.
  • Accesibilidad: semántica, contraste y navegación por teclado.
